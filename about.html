Είμαι ο Θοδωρής, 25 ετών, γεννήθηκα και μεγάλωσα στην Αθήνα και σπούδασα στο ΙΕΚ ΑΛΦΑ. Είμαι Frontend Web Developer, UI/UX Designer και 3D Artist, παθιασμένος με τη δημιουργία σύγχρονων και καθηλωτικών ψηφιακών εμπειριών.

Αγαπώ τη λεπτομέρεια και απολαμβάνω να σχεδιάζω διαδραστικούς ιστότοπους, ελκυστικές διεπαφές χρήστη και εφαρμογές υψηλής απόδοσης. Έχω εμπειρία σε ανάπτυξη ιστού, 3D animation και δημιουργικό σχεδιασμό, με κύρια εργαλεία μου τις JavaScript, CSS, Three.js, React και Blender.

Κάθε νέο project είναι για μένα μια ευκαιρία να πειραματιστώ, να εξελιχθώ και να συνδυάσω την αισθητική με την τεχνολογία. Μου αρέσει να λύνω προβλήματα, να βελτιστοποιώ την απόδοση και να δουλεύω σε καινοτόμες ιδέες.

Αυτή τη στιγμή δημιουργώ το δικό μου 3D Portfolio, το οποίο σύντομα θα ανέβει στο GitHub. Θα έχει μοναδικό design και styling, ιδανικό για έναν Junior Developer που θέλει να ξεχωρίσει.
